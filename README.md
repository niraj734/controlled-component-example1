# React Controlled Input Example
A controlled component is any component whose behavior or rendered output is fully driven by React state, making state the single source of truth.

This is a simple example of a **controlled component** in React.  
The input value is stored in state and updates live as you type.

### How it works:
1. Input value comes from state (`text`).
2. State updates via `onChange`.
